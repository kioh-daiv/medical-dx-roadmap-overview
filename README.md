# 医療DX推進工程表 全体像 — 一般教養向け解説

厚生労働省「医療DXの推進に関する工程表」（第2回医療DX推進本部 令和5年6月2日）を、一般読者向けに俯瞰できるインタラクティブな解説ページに整理したものです。

## 公開ページ

➡ **https://kioh-daiv.github.io/medical-dx-roadmap-overview/**

## 内容

- **基本的な考え方**：5 つの目的の構造（ゴール層 + 手段層 + 横断条件）
- **タイムライン 2023→2030**：年度別の主要マイルストーン
- **4 つの主要施策**：
  1. マイナンバーカードと健康保険証の一体化
  2. 全国医療情報プラットフォームの構築
  3. 電子カルテ情報の標準化等
  4. 診療報酬改定 DX
- **3 文書 6 情報**：電子カルテ情報共有サービスで流通する情報の中身
- **実施主体**：社会保険診療報酬支払基金の抜本改組
- **2030 年のゴール**：到達点と 5 つの目的との対応
